# Gallery / Action Photos

Place real training shots here — adult no-gi rolling, kids class, 8-week beginner course, MMA sparring, Masters class, sauna/cold plunge interior, mat floor, etc.

**Recommended:** Landscape orientation, minimum 1200px wide, portrait 900×1200 for the program cards on the homepage.
**Naming:** descriptive kebab-case — `adults-no-gi.jpg`, `kids-no-gi.jpg`, `beginner-course.jpg`, `mma.jpg`, `masters.jpg`, `recovery.jpg`, `beginner-course-hero.jpg`.

Each image on the site includes a `data-local` attribute pointing to the exact file path. Drop photos in with matching names and run a find-and-replace on the picsum URLs in each HTML file (or run the image-swap helper in the build log).
